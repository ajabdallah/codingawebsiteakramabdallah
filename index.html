About

This website about web design was created by Akram Abdallah as a student project for ARTD 245: Web Design at North Central College in Naperville, IL, in 2025. All the text comes from the following articles located on Dribbble and Flux Academy. The images were from a mix of the actual websites that I used the text from, and my own creations using Photoshop, each image has credits to where it is from. The website has been created using HTML and CSS in VS Code.

Article sources

  • Ultimate Guide To Choosing Colors for Web Design
  • The Ultimate Guide to Choosing Fonts
  • 11 Principles of Design (and How To Use Them)

Project brief

Design and develop a visually pleasing, interactive website with multiple HTML pages, navigation connecting all the pages, featuring a cohesive color scheme, a clear visual hierarchy, a balanced layout, exciting visuals, and text. The goal is to learn how to code/develop an interactive website by applying design principles.
